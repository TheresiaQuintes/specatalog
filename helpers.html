helpers

The package helpers contains some modules that fulfill necessary tasks so that specatalog works properly. Only the module helper_functions is could be interesting for the user, while the other modules are only used internally.

helper_functions

safe_commit(session)

Commit database session.

safe_flush(session)

Flush database session.

make_filter_model(model)

Create dynamically a pydantic-class for filtering based on an SQLAlchemy- model.

make_ordering_model(model)

Create dynamically a pydantic-class for ordering based on an SQLAlchemy model.

make_update_model(model)

Create dynamically a pydantic-class for updating based on an SQLAlchemy model.

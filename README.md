What is specatalog?
-------------------

Specatalog is a tool for the systematic management of an archive with spectroscopic measurement data. It includes the management of a database (containing all metadata) and a directory structure (including raw and processed data).

It can be used by a scripting interface or a GUI.

Download
--------

You can download specatalog via git:

      git clone git@github.com:TheresiaQuintes/specatalog.git
      
Documentation
-------------

The full documentation including advice on installation and usage of the package can be found at:

      https://theresiaquintes.github.io/specatalog/

TTPUpdate

class specatalog.crud_db.update.TTPUpdate(*, name=None, molecular_formula=None, structural_formula=None, additional_info=None, triplet_1=None, linker=None, triplet_2=None, name_suffix=None)

Pydantic update model for TTP. The fields that are set are the parameters that shall be updateted in the database.

The following fields can be selected:

  • name: str

  • molecular_formula: str

  • structural_formula: str

  • additional_info: str

  • triplet_1: Chromophores

  • linker: Linker

  • triplet_2: Chromophores

  • name_suffix: str

Parameters:

  • name (str | None)

  • molecular_formula (str | None)

  • structural_formula (str | None)

  • additional_info (str | None)

  • triplet_1 (Chromophores | None)

  • linker (Linker | None)

  • triplet_2 (Chromophores | None)

  • name_suffix (str | None)

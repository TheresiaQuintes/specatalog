CWEPRUpdate

class specatalog.crud_db.update.CWEPRUpdate(*, temperature=None, solvent=None, concentration=None, date=None, measured_by=None, location=None, device=None, series=None, path=None, corrected=None, evaluated=None, frequency_band=None, attenuation=None)

Pydantic update model for CWEPR. The fields that are set are the parameters that shall be updateted in the database.

The following fields can be selected:

  • temperature: float

  • solvent: Solvents

  • concentration: str

  • date: date

  • measured_by: Names

  • location: str

  • device: Devices

  • series: str

  • path: str

  • corrected: bool

  • evaluated: bool

  • frequency_band: FrequencyBands

  • attenuation: str

Parameters:

  • temperature (float | None)

  • solvent (Solvents | None)

  • concentration (str | None)

  • date (date | None)

  • measured_by (Names | None)

  • location (str | None)

  • device (Devices | None)

  • series (str | None)

  • path (str | None)

  • corrected (bool | None)

  • evaluated (bool | None)

  • frequency_band (FrequencyBands | None)

  • attenuation (str | None)
